# Kafka springcloud config集成

```java
集成步骤：
  1.创建spring-cloud-config-server服务
  2.创建spring-cloud-config-client服务
  3.引入依赖spring-cloud-config-kafka-bus依赖
```

##### spring-cloud-config-client服务application.yaml

```yaml
server:
  port: 7001

logging:
  config: classpath:logback.xml
##动态刷新服务配置
management:
  endpoints:
    web:
      exposure:
        include: health, info, refresh, bus-refresh
##动态刷新配置实例属性
username: luoye
age: 17
spring:
  cloud:
    stream:
      kafka:
        binder:
          zkNodes: 10.211.55.5:2181
          brokers: 10.211.55.5:9092
```

##### spring-cloud-config-client服务bootstrap.yaml

```yaml
spring:
  application:
    name: kafka
  cloud:
    config:
      uri: http://localhost:8900/
      label: master
```
